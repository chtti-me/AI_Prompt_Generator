CH5 AI Prompt 互動提示詞生成系統

Part 1：系統介紹與快速開始

5 分鐘體驗專業級提示詞生成平台

Flask 應用 AI 智能生成 RAG 知識庫

🤔 為什麼需要提示詞生成器？

😰
提示詞工程的挑戰
  • 同樣需求，不同提示詞得到截然不同結果
  • 需要了解 AI 模型特性
  • 不同任務需要不同結構
  • 初學者難以一次寫出完美提示詞
✨
系統化解決方案
  • 互動式問卷引導完善需求
  • 內建模板快速組合提示詞
  • AI 智能生成客製化內容
  • RAG 知識庫累積經驗

🎯 系統四大核心功能

📝
1. 互動式問卷生成器

填寫問卷自動生成專業提示詞，支援模板與 AI 生成兩種模式

📚
2. RAG 知識庫

上傳範本文件，建立個人知識庫，增強 AI 生成品質

⚡
3. 提示詞優化

AI 分析現有提示詞，提供改進建議與優化版本

☁️
4. 雲端協作

同步到 Google Sheets，團隊共同維護提示詞庫

🚀 第一步：啟動專案（30 秒）

Windows 使用者

cd D:\自己架設AI_零基礎到大師\CH5_AI Prompt互動提示詞生成系統
start_windows.bat

macOS / Linux 使用者

cd CH5_AI_Prompt互動提示詞生成系統
chmod +x start.sh
./start.sh
✅ 成功啟動後：終端機會顯示「正在啟動 AI Prompt Generator Platform」以及伺服器位址 http://localhost:5001

🔑 第二步：配置 API Key（1 分鐘）

編輯專案根目錄的 .env 檔案，填入您的 API Key：

使用 Google Gemini（推薦）

GOOGLE_AI_API_KEY=AIzaSy-your-key-here
AI_PROVIDER=gemini
DEFAULT_AI_PROVIDER=gemini

使用 OpenAI

OPENAI_API_KEY=sk-proj-your-key-here
AI_PROVIDER=openai
💡 提示：沒有 API Key 也可以使用系統，只是無法使用 AI 智能生成功能，模板生成功能完全可用。

🏠 第三步：打開系統首頁（10 秒）

打開瀏覽器，訪問 http://localhost:5001

首頁功能：
  • 互動式建置教學頁面，引導完成所有設置步驟
  • 五個主要步驟：環境準備、配置 API Key、初始化資料庫、啟動服務、開始使用
  • 頂端導航列：儀表板、生成器、知識庫、歷史記錄、設定

📝 第四步：體驗提示詞生成器（2 分鐘）

點擊導航列的「生成器」，進入提示詞生成頁面

問卷選項

  • 專案類型：Web 應用、命令列工具、API 服務等
  • 功能需求：前端、後端、資料庫、AI 整合等
  • 詳細程度：基礎、中等、詳細、超詳細
  • 語言選擇：繁體中文、簡體中文、英文
  • 目標平台：跨平台、Windows、Linux、macOS

✨ 生成結果展示

點擊「生成提示詞」按鈕後，系統會產出結構化的 Markdown 格式提示詞

結果包含

  • 專案概述
  • 技術需求
  • 功能規格
  • 資料庫設計
  • API 規劃
  • 前端設計
  • 部署說明

操作按鈕

  • 📋 複製到剪貼板
  • 💾 下載 Markdown
  • ☁️ 同步到 Google Sheets
  • ⚡ 優化提示詞

📚 第五步：建立 RAG 知識庫（1 分鐘）

點擊導航列的「知識庫」，進入 RAG 文件管理頁面

支援格式

  • 純文字檔 (.txt)
  • Markdown 檔 (.md)
  • PDF 文件 (.pdf)
  • Word 文件 (.docx)

語義搜尋

輸入查詢後，系統會在向量資料庫中尋找語義上最相關的內容片段
